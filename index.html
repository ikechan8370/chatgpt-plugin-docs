    ChatGPT-Plugin

    ChatGPT-Plugin For Yunzai-Bot

    使用指南 💡源码

    强大、易用的多AI智能对话插件

    • 在Yunzai根目录下运行 git clone --depth=1 https://github.com/ikechan8370/chatgpt-plugin.git ./plugins/chatgpt-plugin/ 安装插件

    • 执行 pnpm i -P 安装依赖

    • 使用插件自带的后台管理或锅巴插件进行多来源AI的配置

    支持多种大型生成式语言模型

    同时兼具图像生成模型和其他娱乐小功能。更多AI持续接入中……

    OpenAI GPT-3.5/4

    基于OpenAI官方的Chat Completion API进行对话

    • 简单易用

      仅需配置API Key开箱即用

    • 完全掌控

      随心所欲定义模型参数

    • 角色扮演

      通过指定系统消息进行角色扮演

    • 支持代理和反代

      解决因网络等因素无法访问OpenAI API的问题

    OpenAI ChatGPT

    基于OpenAI官方网页版ChatGPT进行对话

    • 完全免费

      与官方保持同步，完全免费

    • 支持GPT-4

      Plus账户可开启GPT-4选项

    • 官网同步

      聊天记录完全同步，每个用户对话隔离

    • 自建反代支持

      提供开源反代部署方法和免费公益反代服务器

    Bing/Sydney

    基于GPT-4的微软AI助手

    • 完全免费

      仅需有效的微软账户，甚至无需登录也可进行聊天

    • Sydney越狱

      提供完全的Jailbreak模式支持，我的Sydney由我自己掌控

    • 无限续杯

      无视官方20条聊天上线的规则限制

    • 内容生成

      支持聊天中的图像生成

    • 多种模式

      原版精准/均衡/创意模式和额外的悉尼与自定义模式

    Claude

    来自Anthropic的Claude assistant，基于Slack Bot

    • 角色扮演

      封装多轮对话以支持自定义角色扮演设定

    • 完全免费

      仅需注册Slack并将Claude加入工作区

    • 更快的速度

      GPT3.5的速度，接近GPT-4的文本质量
