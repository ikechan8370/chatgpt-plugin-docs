    使用指南 💡 仓库

    强大、易用的多AI智能对话插件

    • 安装

      在Yunzai根目录下运行 git clone --depth=1 https://github.com/ikechan8370/chatgpt-plugin.git ./plugins/chatgpt-plugin/ 安装插件

    • 依赖

      执行 pnpm i -P 安装依赖

    • 配置

      使用插件自带的工具箱或锅巴插件进行多来源AI的配置

    • 兼容性

      支持Yunzai、Miao-Yunzai和TRSS-Yunzai (非icqq环境部分功能不可用)

    支持多种多模态大型生成式语言模型

    OpenAI GPT-3.5-turbo/4

    基于OpenAI官方的Chat Completion API进行对话

    • 完全掌控

    • 角色扮演

      通过指定系统消息进行角色扮演

    • 支持代理和反代

      解决因网络等因素无法访问OpenAI API的问题

    • 智能模式

      彻底释放AI潜力，群内禁言、踢人、搜索等，联动AP、喵喵、星铁等其他插件

    OpenAI ChatGPT

    基于OpenAI官方网页版ChatGPT进行对话

    • 完全免费

      与官方保持同步，完全免费

    • 支持GPT-4

      Plus账户可开启GPT-4选项

    • 官网同步

      聊天记录完全同步，每个用户对话隔离

    • 自建反代支持

      提供开源反代部署方法和免费公益反代服务器

    必应/Sydney

    基于GPT-4的微软AI助手

    • 完全免费

      仅需有效的微软账户，支持多账户负载均衡

    • Sydney越狱

      提供完全的Jailbreak模式支持，我的Sydney由我自己掌控

    • 无限续杯

      无视官方20条聊天上线的规则限制，支持pdf等文件解读

    • 内容生成

      支持聊天中的图像生成和图像识别

    • 多种模式

      原版精准/均衡/创意模式和额外的悉尼与自定义模式

    • 辅助验证

      自动应对和通过验证

    Claude

    来自Anthropic的Claude，支持Slack和Claude2

    • 角色扮演

      封装多轮对话以支持自定义角色扮演设定

    • 完全免费

      仅需注册Slack并将Claude加入工作区

    • 更快的速度

      GPT3.5的速度，不一样的文字风格体验

    • 超长上下文

      claude2支持长达100k上下文聊天，支持pdf等文件解读
